查看验证规则

  • Required    验证值输入不为空
  • Min|min    验证输入值不能小于min
  • Max|max    验证输入值不能大于max
  • Range|min|max    验证输入值数值的范围，是否在mil-max区间
  • MinSize|min    验证输入值最小长度
  • MaxSize|max    验证输入值最大长度
  • Length|length    验证输入值指定长度
  • Alpha    验证输入值是否是alpha字符
  • Numeric    验证输入值是否是数字
  • AlphaNumeric    验证输入值是否是 字符或数字
  • Match|string    验证输入值是否能正则匹配 string是正则表达式字符串
  • NoMatch|string    验证输入值是否不能正则匹配 string是正则表达式字符串
  • AlphaDash    验证输入值是否是字符或数字或横杠 -_
  • IP    验证输入值是否是IP 格式，目前只支持 IPv4 格式验证
  • Base64    验证输入值是否是base64 编码
  • Mobile    验证输入值是否是手机号
  • Tel    验证输入值是否是固定电话号
  • Phone    验证输入值是否是手机号或固定电话号
  • ZipCode    验证输入值是否是邮政编码
